Tutorial Semantik Elemen untuk Pemula

Secara umum teradapat kurang lebih 100 elemen semantic. Namun berikut elemen semantik sering ditemui :

  1. article —> untuk membuat elemen artikel
  2. aside —> untuk membuat elemen bagian samping
  3. details —> untuk membuat elemen datail atau spoiler
  4. figcaption —> untuk membuat teks caption pada figure
  5. figure —> untuk membuat figur atau gambar pada artikel
  6. footer —> untuk membuat elemen bagian kaki dari web
  7. header —> untuk mebuat kepala kop dari web
  8. main—> untuk membuat elemen utama
  9. mark —> untuk menandai teks
  10. nav —> untuk membuat navigasi
  11. section —> untuk membuat bagian artikel
  12. summary —> untuk membuat ringkasan artikel atau isi spoiler
  13. time —> untuk membuat elemen yang menyatakan waktu
